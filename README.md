## 🚀 Quick Deploy

### Prerequisites
1. GitHub account
2. Vercel account (free tier)
3. Supabase account (free tier)

### One-Click Setup

```bash
# 1. Clone
git clone git@github.com:landyourweb-hq/landyourweb.com.git
cd landyourweb.com

# 2. Install
npm install

# 3. Configure
cp .env.example .env.local
# Fill in your Supabase keys from https://supabase.com/dashboard

# 4. Run locally
npm run dev

# 5. Deploy
npx vercel --prod
```

### Database Setup

Run the schema file in Supabase SQL Editor:
```
supabase/schema.sql
```

This creates tables for:
- **contacts** — website form submissions
- **leads** — Mercury pipeline output
- **analytics_events** — custom event tracking
- **subscribers** — newsletter signups
- **projects** — client project management
